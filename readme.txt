== Description ==

This plugin allows customers to signup and signin in <a href="https://learntalk.org/" target="_blank" title="learntalk.org">learntalk.org</a> using the API. 
Learntalk is an on-demand language school. their aim is to democratize access to high quality English teachers through video-call lessons. and this plugin gives functionality to signup and sign in using learntalk API

In order for you to gain access to the APIs, you and your organisation must meet the following pre-requisites:

1.Your organisation must have an active API of Signup and Sign in.
2.You must have Authentication Tokan for API.
3.You must have reviewed and accepted the <a href="https://learntalk.org/en/terms-of-service">Terms and Conditions </a>for use of APIs.

= Features: =

* Auto login after signup
* Can be add dynamic API of Learntalk organisation in setting api page 
* For signup form use this short-code - [lrnts_signup]
* For Sign in form use this short-code - [lrnts_signin] 
* Forgot password form 
* Supports the latest version of Wordpress

== Installation ==

To install Learntalk Signup, follow these steps:

1. Download and unzip the plugin
2. Upload the entire learntalk-signup/ directory to the /wp-content/plugins/ directory
3. Activate the plugin through the Plugins menu in WordPress
